PRISM Workshop On High Performance Computing

This tutorial covers an introduction to using High Performance Computing (HPC) cluster at Ohio State, focusing on how to use the Unity cluster. The Unity cluster is free for ASC students at Ohio State, however, many students may have questions about working in an HPC enviroment. I provide a simple example and documentation to get started with R on the unity cluster here: Intro to Using Unity Cluster
